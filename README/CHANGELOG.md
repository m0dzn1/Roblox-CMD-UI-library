# Changelog

All notable changes to CMD UI Library will be documented in this file.

## [1.0.0] - 2026-02-05

### Added
- Initial release of CMD UI Library
- Rainbow ASCII art boot screen with "CMD UI made by m0dzn"
- Dynamic loading animation (randomized 1.000-2.000 seconds)
- Accurate load time display
- Draggable CMD window interface
- Classic Windows CMD styling (black background, green text, blue title bar)
- `SetTitle()` - Change window title dynamically
- `Print()` - Output text to console with optional color
- `SelectByNumber()` - Numbered menu selection
- `Confirm()` - Yes/No prompt system
- `Input()` - General text/number input
- `Clear()` - Clear console content
- `WaitForReady()` - Wait for boot sequence completion
- `Destroy()` - Close and remove UI
- Auto-scrolling console output
- Input validation for numbers
- Error handling and user feedback
- Code font (closest to Lucida Console)
- Comprehensive documentation
- Example scripts and use cases

### Features
- 100% accurate Windows CMD appearance
- Rainbow gradient ASCII art with animation
- Responsive and user-friendly interface
- Lightweight and performant
- Easy to integrate into any Roblox project
- Fully customizable text colors
- Professional error handling

### Examples Included
- Basic usage example
